## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Run E2E tests

Start backend express server and then 

```
npm run e2e
```

### Lints and fixes files
```
npm run lint
```
